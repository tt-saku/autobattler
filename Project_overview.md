项目大纲
项目名称：大巴扎魔改自走棋/autobattler_TheBazaar_mod
项目开发引擎：Godot 4.6.3
项目开发语言：GDScript

一、核心玩法概述
这是一款异步自走棋游戏，魔改于《大巴扎》《The Bazaar》。玩家在战前通过布置物品（卡牌）到10个线性槽位上，并设置每个槽位的正面（A面）或背面（B面），来影响战斗中物品触发的效果。
每张物品卡拥有A面效果和B面效果（未来可扩展更多面），效果强度不同：A面偏保守，B面偏激进。
物品有大小（小=1格，中=2格，大=3格），占多格时根据最外侧槽位的面决定整张卡当前生效的面。
战斗中，物品通过冷却（CD）系统自动触发效果，并且会因槽位面的变化而发生翻面，从而改变后续效果。

二、槽位系统（核心机制）
2.1 槽位布局
共有10个线性槽位，从左到右编号 0~9。
槽位面：每个槽位有一个面状态（A面 或 B面）。
槽位索引	面规则
0（最左）	固定B面（玩家准备环节不可手动改，但战斗内可被改变）
9（最右）	固定A面（同上）
1~8	玩家可在战斗前自由设置A/B（手动点击切换）

2.2 物品占格与生效面规则
小型卡：占1格 → 直接使用该槽位的面。
中型卡：占2格 → 取占用的两个槽位中更靠近外侧的槽位的面（距离计算：左槽距离左端，右槽距离右端，取距离更近者）。
*中型卡特殊居中情况：若占用的两个槽位正好对称居中（例如占槽5和6），则默认以左侧槽（5号）的面为准。
大型卡：占3格 → 同样取占用的三个槽位中最靠外侧的槽位的面（左端或右端距离最小者）。
所有物品在放置时，UI会实时预览其当前生效的面。

三、战斗流程与CD机制
3.1 阶段划分
准备阶段（无限时间，无战斗）
玩家可以：
拖拽物品到任意空槽位（不可重叠，占格冲突自动阻止）。
点击槽位1~8切换其A/B面。
查看每个物品的当前面、效果描述、CD等信息（建议使用鼠标悬浮栏）。
所有操作瞬时生效，无硬直、无CD。

战斗阶段（自动进行，不可操作）
根据战前布局自动运行物品逻辑。
战斗中可以发生：
物品自动释放技能（基于CD）。
某些稀有词条改变槽位面（影响后续翻面）。
翻面发生（由CD结束后的检查触发）。

3.2 物品冷却（CD）
每个物品拥有独立技能CD，最小值 1秒。
CD结束后，物品会：
执行多重触发（若有，例如一次技能产生多次效果，如连击）。
所有多重触发效果间隔 0.25 秒依次执行（串行队列，防止瞬间爆炸）。
多重触发全部完成后，进行翻面检查（见第四章）。
开始下一次 CD（长度为物品自身的CD值，≥1秒）。
多重触发的例子：一个技能效果为“造成50伤害，多重触发：2”，则两次伤害效果之间等待0.25秒。

3.3 槽位改变的硬直
战斗中，任何改变槽位面的效果（由物品词条触发）自带 1秒硬直。
即：一个槽位被改变后，1秒内无法再次被任何效果改变（但可以被多个不同效果排队等待，遵循间隔）。
硬直按单个槽位独立计算，不影响其他槽位。
硬直期间，槽位面保持不变。

四、翻面机制（核心策略）
4.1 触发时机
翻面是一个被动检查，发生在：
物品CD结束 → 多重触发执行完毕 → 即将进入下一次CD之前。
检查条件：物品当前的面 vs 它所在槽位的当前面。
若不一致 → 物品立即翻面（变为槽位的面），并触发一次翻面效果（由卡牌自身定义）。
若一致 → 不翻面，直接进入下一次CD。

4.2 翻面效果
每张卡有独特的翻面效果，例如：
永久增加自身攻击力/最大生命值等。
改变某槽位的面（需遵守硬直规则）。
对敌方造成伤害、影响敌方构筑等。
翻面效果默认是被动，如果涉及多次触发会有0.25s硬直。但为设计时遵守强度设计，避免滥用。

4.3 错面战斗
如果物品CD较长（如3秒），在CD期间其所在槽位被其他效果改变，则物品会保持原本的面战斗，直到CD结束才翻面。
这是故意设计的策略点：玩家可预判对方的长CD物品，通过改变槽位使其暂时“错面”，改变其收益。

五、词条与稀有性
改变槽位的词条：相对稀有（参考大巴扎里的摧毁、转化词条），且消耗较大。
翻面相关combo：允许设计，但原理上避免：
单个物品无依赖地左脚踩右脚（例如“翻面后增加攻击，增加攻击后又立即翻面”）。
容易触发的combo有太高的成长性（例如每次翻面加5攻击，且每2秒就能翻一次）。

六、UI与信息反馈要求
要素	展示方式
槽位当前面	每个槽位底下显示A/B图标，对应蓝/红的槽位边框，点击底下图标切换，固定边界槽有锁标志（但战斗内就会解锁）
物品当前面	物品设计两个牌面，对应A/B面，自动匹配其当前放置的槽位（无法手动翻面），与槽位面不一致时可以用颤抖等动态效果提示其将翻面
冷却条	参考大巴扎，时牌面从下往上的光束进度条
战斗日志	可展开的侧边栏，记录每时间槽位变化、翻面事件、技能触发等一切战斗信息。通过这些信息能够完全复原、回放战斗
硬直效果	槽位被改变后，1s硬直内短暂使其槽位颜色发光/高亮

七、实现建议
一切以大巴扎为基准，以下建议均为目前暂定：
物品本身在图鉴需要：ID、名称、A/B面的牌面图绘、型（小中大）、各个稀有度的{A/B面的效果、翻面的效果}、CD（秒,A/B面）、tag（比如武器、伙伴、工具等）、可以出现在的商人/事件、各个附魔后的效果等
物品在游戏中是相互独立的（敌方的、我方的、复制的等），需要独立记录：当前面、当前tag（会临时加）、当前价值、当前CD（A/B面）、当前各个效果（包括被附魔的）、当前暴击率、当前状态（加减速、冰冻、摧毁、飞行）等。

八、开发完整度预计
大约100张卡、游玩体感接近大巴扎本体，美术资源可以简约一些，战斗相关核心必须完整且成熟

九、附加说明
未来扩展可能性：
增加更多面（C面等特殊面）、增加职业（目前就没有职业区分，以后对标大巴扎做出职业差异）、利用AI做平衡与建设新构筑。

除此之外的不确定的要求都参考大巴扎《The Bazaar》
